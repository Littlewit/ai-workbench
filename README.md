# ai-workbench
一款集成 AI 智能聊天、RAG 文档问答、文案生成、代码助手的全栈 AI 应用，前端负责交互与流式渲染，Node 中间层实现 API 安全代理，整体可直接部署上线，聚焦前端视角的 AI 应用开发，兼顾实用性与可扩展性
## 核心功能（面试重点）

- **AI 智能聊天**：基于通义千问 qwen-turbo 模型，实现流式输出（打字机效果），支持多轮对话，聊天记录本地持久化，提升用户交互体验。

- **RAG 文档问答**：支持 TXT 文档上传，实现“文档读取-内容解析-AI 精准问答”全流程，仅基于上传文档内容回复，核心体现 RAG 基础实现逻辑。

- **文案生成工具**：支持朋友圈、产品介绍、工作总结等多场景文案生成，一键触发，流式返回结果，满足日常文案需求。

- **AI 代码助手**：支持 Vue/JS/TS 等前端代码生成、代码解释、问题排查，返回简洁规范的代码及说明，适配前端开发场景。

- **安全代理机制**：通过 Node.js + Express 搭建中间层，隐藏 API Key，避免前端暴露密钥，实现跨域处理，保障接口安全，支持线上部署。
